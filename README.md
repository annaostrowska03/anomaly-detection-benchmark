# Anomaly Detection Benchmark

This repository contains a comprehensive academic project focused on Unsupervised Anomaly (Outlier) Detection. It evaluates multiple machine learning algorithms and implements a robust ensemble approach to identify anomalies in unlabeled data.

Author: [Anna Ostrowska](https://github.com/annaostrowska03)

This project was developed as an assignment for the "Data Exploration and Visualization" Data Science course @ WUT.

## Project Overview

The main objective of this project is to evaluate the effectiveness of various outlier detection methods and build a reliable pipeline for a verification task on an entirely unlabeled, 21-dimensional dataset. Because true labels are missing, the core solution utilizes an Ensemble Majority Voting system combining three distinct mathematical approaches to minimize false positives and capture both local and global anomalies.

### Algorithms Evaluated & Discussed:

- Isolation Forest (Tree-based)

- Local Outlier Factor - LOF (Density/Distance-based)

- One-Class SVM (Boundary/Kernel-based)

- DBSCAN (Clustering-based)
